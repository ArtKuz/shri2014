  13. Вопрос:
    Пользуетесь ли вы командной строкой? Какие команды используете и какие задачи решаете с их помощью? С какими программами преимущественно взаимодействуете через интерфейс командной строки?
    Ответ:

    Да, я пользуюсь командной строкой.
    git - для контроля версий проекта.
    mc - для скорейшего перехода между файлами.
    ssh - для подключения к удаленным серверам.
    scp - для копирования файлов между удаленными серверами.
    mysql - для работы с "MySQL".
    tar - для архивирования.
    top, atop, kill, ps aux - для работы с процессами системы.
    grep -lr - для поиска файлов с конкретным содержанием.
    du -sh, df -h - для определения занимаемого мета на жестком диске теми или иными данными.
    subl, nano, vim - для редактирования файлов.
    php, bash, sh - для исполнения файлов.
    npm - для установки модулей/плагинов.
